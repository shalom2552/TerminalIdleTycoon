## Usage
1. Make sure python3 is installed on your machine
```bash
sudo apt-get -y install python3 
python3 --version
```
2. Go to file location
```bash
cd ~/{your_download_folder}/TerminalIdleTycoon/
```
3. Run the script
```bash
python3 ./IdelTycoon.py
```
